# BlinkIT-Grocery
This project analyzes BlinkIT grocery sales data using SQL, Excel, and Power BI. It includes data cleaning, transformation, and interactive dashboard creation to track sales trends, customer behavior, and product performance. The goal is to generate actionable insights for better business decision-making

🛒 BlinkIT Grocery Data Analysis
📌 Project Overview

This project focuses on analyzing BlinkIT grocery sales data to extract meaningful insights and support data-driven decision-making. It covers the complete data analysis workflow including data cleaning, transformation, and dashboard visualization.

The analysis helps understand sales performance, customer behavior, and product trends in the grocery delivery business.

🎯 Objectives
Analyze overall sales performance
Identify top-performing product categories
Understand customer ratings and preferences
Compare outlet performance based on size and location
Generate actionable business insights
🛠️ Tools & Technologies
SQL – Data extraction and querying
Excel – Data cleaning and preprocessing
Power BI – Dashboard and data visualization
Python (optional) – Data analysis and EDA
📂 Dataset Information

The dataset contains grocery sales and outlet details such as:

Item Type

These features help in evaluating performance metrics and trends.
